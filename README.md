# projetoBaseDados_2020.2021

***

## Realização de uma base de dados baseado num sistema de avaliação de filmes.

### **Utilização:**

*Os ficheiros .sql tem o objetivo de ser utilizados com o programa MySQL Workbench*

Ordem de execução dos ficheiros:

1. tabelas.sql

   Cria a base de dados e as respetivas tabelas;

2. registos.sql

   Adiciona registos e informação á base de dados;

3. consultas.sql

   Cria vistas e realiza vários tipos de consultas á base de dados;

4. utilizadores.sql - *Necessita de um server conectado ao MySQL Workbench*

   Cria *users*, neste caso um *admin* e um *cliente*.
